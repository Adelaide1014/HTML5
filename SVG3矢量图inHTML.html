SVG 文件可通过以下标签嵌入 HTML 文档：embed、object 或者 iframe


使用 embed标签:
embed标签被所有主流的浏览器支持，并允许使用脚本。
当在 HTML 页面中嵌入 SVG 时使用 embed标签是 Adobe SVG Viewer 推荐的方法！
然而，如果需要创建合法的 XHTML，就不能使用 embed。任何 HTML规范中都没有 embed标签。
pluginspage 属性指向下载插件的 URL。
使用 object标签:
object 标签是 HTML4 的标准标签，被所有较新的浏览器支持。它的缺点是不允许使用脚本。
codebase 属性指向下载插件的 URL。
